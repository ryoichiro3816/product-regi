## 💬 Usage

👉https://hub.docker.com/editions/community/docker-ce-desktop-mac (Install docker for mac)

`$ https://github.com/ryoichiro3816/product-regi.git`

`$ cd product-regi`

`$ docker-compose up -d`

`$ docker-compose exec web bash`

`$ rails db:create`

`$ rails db:migrate
`
`$ rails s -b 0.0.0.0`

👉 http://localhost:3000
